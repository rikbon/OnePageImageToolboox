# Getting Started with 1ps Image Toolbox

Welcome to the **1ps Image Toolbox**, a suite of private, client-side tools for all your image manipulation needs.

## Why 1ps?
- **Privacy First**: All processing happens in your browser. No images are ever uploaded to a server.
- **Fast**: Built on modern web technologies for instant results.
- **Easy**: Intuitive interface with drag-and-drop support.

## Required Software
- A modern web browser (Chrome, Firefox, Edge, Safari).
- To run locally: [Docker](https://www.docker.com/) OR [Node.js](https://nodejs.org/).

## Running the Application
### Option 1: Docker (Recommended)
1. Clone the repository.
2. Run `docker compose up --build`.
3. Open `http://localhost:8080`.

### Option 2: Node.js
1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Run `npm run dev` to start the development server.
4. Open the URL provided in the terminal (usually `http://localhost:5173`).
