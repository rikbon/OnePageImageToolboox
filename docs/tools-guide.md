# Tools Guide

Here is a breakdown of the available tools in the 1ps application.

## 1. Resize Tool
Simply resize an image by width or height.
- **Features**: Aspect ratio locking, pixel dimensions.
- **Usage**: Upload image -> Enter new width/height -> Download.

## 2. Converter Tool
Convert images between popular formats.
- **Supported Formats**: JPEG, PNG, WEBP.
- **Usage**: Upload image -> Select output format -> Download.

## 3. Crop Tool
Crop unwanted parts of an image.
- **Features**: Interactive cropping box, aspect ratio presets.
- **Usage**: Upload -> Drag crop handles -> Download.

## 4. Rotate & Flip
Fix orientation issues.
- **Features**: 90-degree rotations, Horizontal/Vertical flipping.

## 5. Filters
Apply artistic filters to your photos.
- **Available Filters**: Grayscale, Sepia, Invert, Blur, Brightness, Contrast.

## 6. Text Overlay
Add simple text annotations.
- **Features**: Custom text, position, color, and size.

## 7. Color Adjustments
Fine-tune image colors.
- **Controls**: Brightness, Contrast, Saturation, Hue.

## 8. Bulk Resize
Resize multiple images at once.
- **Workflow**: Upload multiple files -> Set dimensions -> Download all as individual files.

## 9. Compressor
Reduce file size for web use.
- **Features**: Quality slider, live file size preview.

## 10. Base64 Converter
Developer utility for data URIs.
- **Image to Base64**: Get the string representation of an image.
- **Base64 to Image**: Visualize a Base64 string.

## 11. Meme Generator
Create memes instantly.
- **Features**: Classic Top/Bottom text layout, custom font colors.
